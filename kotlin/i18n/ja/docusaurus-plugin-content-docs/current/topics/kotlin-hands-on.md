Kotlinを用いて、さまざまなテクノロジーを使用し、複数のプラットフォームをターゲットとするアプリケーションを作成できる一連のハンズオンチュートリアルです。
演習は一連のステップに分かれており、各セクションを順を追って説明します。

## Spring BootとKotlinを使用したWebアプリケーションの構築
