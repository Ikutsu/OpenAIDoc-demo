[Kotlin Notebook](kotlin-notebook-overview) 是一個互動式工具，讓您可以在一個文件中混合程式碼、視覺效果和 markdown。
您可以使用 Notebook 在稱為「程式碼儲存格」（_code cells_）的區段中編寫和執行程式碼、立即查看結果，並寫下您的想法。
這種設定使其成為快速原型設計、分析和資料科學的絕佳工具。
