# Ktor

Ktor 是一个用 Kotlin 开发的异步 Web 框架，专为构建互联应用程序而设计。它可用于创建服务器端和客户端应用程序，并支持各种协议，如 HTTP、WebSockets 等。

## 核心特性

- **协程支持** - 基于 Kotlin 协程的非阻塞异步设计
- **模块化** - 通过插件系统实现灵活的功能扩展
- **多平台** - 可在 JVM、JavaScript 和 Native 平台中使用
- **轻量级** - 小巧且可扩展的核心框架
- **类型安全** - 利用 Kotlin 强大的类型系统实现的 API 安全性
